<a name="installation"></a>
## Installation
Install the `node_modules` using `npm`:

```bash
npm install
```

<a name="development-server"></a>
## Development Server
Run this command:

```bash
ng serve --live-reload false
```

After successfully running the development server, navigate to:

```bash
http://localhost:4200/
```
